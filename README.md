# Claude Skills Collection

This repository contains custom skills for Claude Code.

## Skills Included

- **claude-agent-ts-sdk** - Build Claude agents using TypeScript
- **docx** - Document creation, editing, and analysis
- **pdf** - Comprehensive PDF manipulation toolkit
- **pptx** - Presentation creation, editing, and analysis
- **skill-creator** - Guide for creating new skills
- **svelte-5-runes** - Complete guide for Svelte 5 runes
- **webapp-testing** - Toolkit for testing local web applications
- **worldcat-api** - Use OCLC WorldCat APIs for bibliographic data
- **xlsx** - Spreadsheet creation, editing, and analysis

## Installation

To use these skills in Claude Code, copy the skill directories to your `~/.claude/skills/` directory.

## Usage

Skills are automatically loaded by Claude Code from the `~/.claude/skills/` directory.
